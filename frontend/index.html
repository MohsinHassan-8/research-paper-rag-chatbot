How it works

From question to grounded answer

index

Documents are chunked & embedded

Papers are split into overlapping passages and embedded with a sentence-transformer model, then stored in a vector database.

ask

You ask a question

Your question is embedded the same way, so it can be compared directly against every indexed passage.

retrieve

Relevant passages surface

MMR retrieval pulls the most relevant, least redundant passages per source document, not just the closest match.

answer

An answer is written, with citations

The model answers using only the retrieved passages, and every source file and page number is shown underneath.
